# FastPostProcessing
A small and fast post processing stack targeting mobile and mobile VR for Unity. Everything is done in only one pass.

## Effects
- Sharpen
- Bloom
- Tonemapping (ACES, Dawson, Hable, Photographic, Reinhart)
- LUT Texture
- Dithering
- Gamma Correction
