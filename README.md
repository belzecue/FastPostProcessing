# FastPostProcessing
A small and fast post processing stack targeting mobile and mobile VR for Unity.

## Effects
- Fast Bloom with many parameters
- Tonemapping (ACES, Dawson, Hable, Photographic, Reinhart)
- LUT Texture support
- Dithering
- Gamma Correction
